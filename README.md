# 🏠 House Price Prediction using Machine Learning

## 📌 Overview
This project predicts house prices based on features like area, number of bedrooms, and location using Machine Learning.  
It is built with a Flask web application and provides real-time predictions along with visualization.

---

## 🚀 Features
- Predict house prices instantly
- Location-based prediction 📍
- Machine Learning model (Random Forest)
- Accuracy display (R² Score)
- Graph visualization 📊
- Clean and responsive UI

---

## 🛠 Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas, NumPy
- Matplotlib
- HTML, CSS

---

## 📊 Machine Learning Model
- Algorithm: Random Forest Regressor
- Problem Type: Regression
- Evaluation Metric: R² Score
